翻訳デモ用として、シンハラ語（スリランカの公用語）の短文フレーズを50パターン作成しました。
これらは、ワーカーが管理者にチャットで送ってくる「日常の報告」「体調管理」「現場での質問」などを想定しています。デモ画面では、**「左側にシンハラ語、右側にAIによる日本語訳」**を表示させることで、言語の壁を越えた管理の利便性を一気にアピールできます。
AI翻訳デモ用：シンハラ語フレーズ集（50パターン）
※形式：【シンハラ語（原文）】 / （読み） / [日本語訳]
1. 挨拶・基本（基本のやり取り）
 * Subha udayak. / (スバ ウダヤッ) / [おはようございます]
 * Subha rathriyak. / (スバ ラートリヤッ) / [おやすみなさい]
 * Isthuthi. / (イストゥティ) / [ありがとうございます]
 * Sama wenna. / (サマ ウェンナ) / [すみません/ごめんなさい]
 * Ow. / (オウ) / [はい]
 * Na. / (ナ) / [いいえ]
 * Kohomada? / (コホマダ) / [お元気ですか？/どうですか？]
2. 勤怠・体調（最も重要な管理項目）
 * Ada mata enna baha. / (アダ マタ エンナ バハ) / [今日は行けません]
 * Mata una. / (マタ ウナ) / [熱があります]
 * Tikak pama wei. / (ティカッ パマ ウェー) / [少し遅れます]
 * Bus eka pama wei. / (バス エカ パマ ウェー) / [バスが遅れています]
 * Mata asaneepai. / (マタ アサニーパイ) / [体調が悪いです]
 * Ada niwaduwak ganna puluwanda? / (アダ ニワドゥワッ ガンナ プルワンダ) / [今日休みをもらえますか？]
 * Ude nagitte pama wela. / (ウデー ナギッテ パマ ウェーラ) / [朝、起きるのが遅れました（寝坊しました）]
 * Kassa thiyanawa. / (カッサ ティヤナワー) / [咳が出ます]
 * Bada ridenawa. / (バダ リデナワー) / [お腹が痛いです]
 * Oluwa ridenawa. / (オルワ リデナワー) / [頭が痛いです]
3. 現場での質問・報告（業務連絡）
 * Meka karanne kohomada? / (メーカ カランネ コホマダ) / [これはどうやるのですか？]
 * Mata therenne naha. / (マタ テレンネ ナハ) / [私には分かりません]
 * Tawa ekak ona. / (タワ エカッ オナ) / [もう一つ必要です]
 * Weda iwarai. / (ウェーダ イワライ) / [仕事が終わりました]
 * Wadi pura weda thiyanawada? / (ワディ プラ ウェーダ ティヤナワーダ) / [残業はありますか？]
 * Meka hariyada? / (メーカ ハリヤダ) / [これで合っていますか？]
 * Yanthraya nathera una. / (ヤントラヤ ナテラ ウナ) / [機械が止まりました]
 * Gloves iwarai. / (グラブス イワライ) / [手袋がなくなりました]
 * Ada kiyatada iwara wenne? / (アダ キヤタダ イワラ ウェンネ) / [今日は何時に終わりますか？]
 * Mata udaw ona. / (マタ ウダウ オナ) / [助けが必要です]
 * Sapa parippu na. / (サパ パリップ ナ) / [（専門用語/隠語）作業が間に合いません/大変です]
4. 生活・寮・給与（トラブルの芽）
 * Wathura naha. / (ワトゥラ ナハ) / [水が出ません]
 * Internet weda naha. / (インターネット ウェーダ ナハ) / [インターネットが繋がりません]
 * Kamaraya rath wenawa. / (カマラヤ ラトゥ ウェナワー) / [部屋が暑いです]
 * Light weda naha. / (ライト ウェーダ ナハ) / [電気がつきません]
 * Padi labune naha. / (パディ ラブネ ナハ) / [給料が届いていません]
 * Payslip eka ona. / (ペイスリップ エカ オナ) / [給与明細が必要です]
 * Gas iwarai. / (ガス イワライ) / [ガスがなくなりました]
 * Supermarket eka koheda? / (スーパーマーケット エカ コヘダ) / [スーパーはどこですか？]
5. ビザ・書類・手続き（管理会社のメイン業務）
 * Visa eka labunada? / (ウィザ エカ ラブナダ) / [ビザは取れましたか？]
 * Residence card eka nathi una. / (レジデンス カード エカ ナティ ウナ) / [在留カードを失くしました]
 * Mata letter eka ona. / (マタ レター エカ オナ) / [書類（証明書）が必要です]
 * Passport eka thiyanawa. / (パスポート エカ ティヤナワー) / [パスポートを持っています]
 * Inshuwaranse eka gena kiyanna. / (インシュアランス エカ ゲナ キヤンナ) / [保険について教えてください]
6. AIへの相談・その他（自由入力想定）
 * Mata lankawata yanna ona. / (マタ ランカワタ ヤンナ オナ) / [スリランカに帰りたいです]
 * Kauruhari mata benna. / (カウルハリ マタ ベンナ) / [誰かに怒られました/悪口を言われました]
 * Meka godak barai. / (メーカ ゴダッ バライ) / [これはとても重いです]
 * Heeta enna puluwanda? / (ヘタ エンナ プルワンダ) / [明日来てもいいですか？]
 * Mata mahansiyi. / (マタ マハンシイ) / [疲れました]
 * Kema hoda naha. / (ケーマ ホーダ ナハ) / [食べ物が合いません]
 * Godak isthuthi udawwata. / (ゴダッ イストゥティ ウダウワタ) / [助けてくれて本当にありがとう]
 * Mata katha karanna puluwanda? / (マタ カタ カランナ プルワンダ) / [お話しできますか？]
 * Prashnayak thiyanawa. / (プラシュナヤッ ティヤナワー) / [問題があります]
デモでの演出方法のアドバイス
 * チャット風UI: 管理者のダッシュボードに「未読メッセージ（5件）」のようなバッジを出し、クリックすると上記のシンハラ語が並んでいる。
 * 翻訳ボタン: 各メッセージの横に「AI翻訳」ボタンを配置。押すとふわっと日本語に切り替わる演出。
 * 感情分析 (Sentiment Analysis): 43番（怒られた）や42番（帰りたい）などのネガティブなメッセージには、AIが**「要注意（赤色）」**のタグを自動で付与するデモにすると、管理者の食いつきが良くなります。
この50パターンがあれば、かなりリアリティのある管理画面デモが構築できそうですね！
